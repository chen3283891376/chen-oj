# 第一次尝试

## 题目背景

小C想去轰炸FCC总部，TA带来一个二手市场淘来的`金箍棒手雷`，请你帮助TA完成这项艰巨的任务。

## 题目描述


已知小C的手雷可以扔 $n$ 米，TA距离FCC总部有 $m$ 米（$|m|,|n| \le {10}^9$）。
请问你要前进几米，才可以炸毁FCC总部。

## 输入格式

第一行，一个整数 $T$（$|T| \le {10}^9$）
下面 $T$ 行，每行是两个以空格分开的整数，即 $n,m$。

## 输出格式

T个整数

## 样例 #1

### 样例输入 #1

```
3
5 8
10 12
20 15
```

### 样例输出 #1

```
3
2
-5
```
